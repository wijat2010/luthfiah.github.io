  • Nama saya Fahimah Luthfiyyah, biasa di panggil Luthfiah. Saya tinggal di Bekasi barat. Saya adalah anak kedua dari dua bersaudara, saya memiliki saudara kandung laki-laki. saya terlahir di keluarga yang sederhana. Saya memiliki hati yang sangat perasa, serta jujur. Ketika berumur 6 tahun, saya memulai pendidikan di SDN Bintara Jaya 1, Bekasi. Kemudian setelah lulus saya melanjutkan pendidikan di MTsN 21 Jakartaj lalu dilanjutkan dengan pendidikan Andi MAN 18 Jakarta dengan jurusan IPS. Ketika dibangku SMA, saya mengikuti ekstrakurikuler jurnalistik dengan divisi fotografi, saya selalu menjadi bagian dari team dokumentasi sekolah dari ekstrakurikuler jurnalistik. Hasil foto saya cukup baik untuk dokumentasi kegiatan sekolah, dan saya sangat menyukai tugas saya di jurnalistik.

  • Tak hanya fotografi, saya juga belajar mengenai cinematography secara otodidak. Saya juga menyukai desain, meskipun tidak hobi. Namun, ketika di kelas 2 SMA, penglihatan saya mulai menurun sehingga saya sudah tidak aktif lagi di jurnalistik hingga kelas 3 SMA dan hanya fokus dengan ujian sekolah. Setelah lulus, saya memeriksakan kondisi mata saya ke ahli mata, doktermengatakan bahwa penyebab mata saya yaitu glukoma dengan faktor genetik. Saat ini saya mahasiswa semester 3 dengan program studi ilmu perpustakaan di UIN Syarif Hidayatullah Jakarta. Saya berkeinginan untuk lulus tepat waktu meskipun Dengan kondisi saya yang sangat terbatas.
